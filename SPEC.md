# 学习精灵 (Spirit Pet) - AI宠物学习应用规格

## 1. 项目概述

**项目名称**: 学习精灵 (Spirit Pet)  
**项目类型**: Web应用 (React + TypeScript)  
**核心功能**: 一个寓教于乐的AI宠物养成平台，学生通过学习行为培养和进化自己的"学习精灵"  
**目标用户**: 学生群体（K12阶段）

---

## 2. UI/UX 规格

### 2.1 布局结构

**页面组成**:
- **Header**: 固定顶部，包含应用Logo、宠物名称、能量条
- **主区域**: 宠物展示区、互动控制区、状态信息区
- **侧边栏**: 任务面板、成就系统（可收起）

**响应式断点**:
- 移动端: < 768px (单列布局)
- 平板: 768px - 1024px (双列布局)
- 桌面: > 1024px (三列布局)

### 2.2 视觉设计

**色彩方案**:
- 主色: `#6C5CE7` (神秘紫 - 代表智慧)
- 次色: `#00CEC9` (青色 - 代表成长)
- 强调色: `#FDCB6E` (金色 - 代表能量)
- 背景色: `#1A1A2E` (深空蓝黑)
- 卡片背景: `#16213E` (深海蓝)
- 文字主色: `#FFFFFF`
- 文字次色: `#A0A0B0`

**字体**:
- 标题: "ZCOOL KuaiLe", sans-serif (可爱活泼)
- 正文: "Noto Sans SC", sans-serif
- 数字/状态: "Orbitron", monospace

**间距系统**:
- 基础单位: 8px
- 小间距: 8px
- 中间距: 16px
- 大间距: 24px
- 超大间距: 32px

**视觉效果**:
- 卡片: 圆角16px, 微妙阴影 `0 4px 20px rgba(108, 92, 231, 0.3)`
- 按钮: 圆角12px, 悬停发光效果
- 宠物区域: 动态粒子背景, 呼吸光效
- 进化动画: 闪光粒子爆发 + 形态变换过渡

### 2.3 组件列表

1. **PetDisplay** - 宠物主展示组件
   - 3D/2D宠物模型显示
   - 呼吸动画
   - 情绪状态可视化

2. **EnergyBar** - 能量条组件
   - 显示进化能量
   - 分段式显示不同属性能量

3. **ChatPanel** - 对话面板
   - 消息气泡
   - 输入框
   - 快捷回复按钮

4. **TaskCard** - 任务卡片
   - 任务类型图标
   - 进度指示
   - 奖励预览

5. **EvolutionPanel** - 进化展示面板
   - 进化方向选择
   - 预览图
   - 进化条件说明

6. **StatsPanel** - 属性统计面板
   - 理性值 (数学)
   - 诗意值 (语文)
   - 探索值 (科学)
   - 社交值 (英语)

## 3. 功能规格

### 3.1 核心功能

#### 3.1.1 宠物养成系统
- **初始形态**: 蛋形态 → 幼年期 → 成长期 → 成熟期
- **属性系统**:
  - 理性值: 通过数学学习提升 → 解锁"理性翅膀"
  - 诗意值: 通过语文学习提升 → 解锁"诗意光环"
  - 探索值: 通过科学学习提升 → 解锁"探索之眼"
  - 社交值: 通过英语学习提升 → 解锁"沟通天赋"
- **进化机制**: 能量达到阈值可选择进化方向

#### 3.1.2 AI对话系统
- **对话场景**:
  - 学习鼓励: 检测到学习行为时
  - 关心提醒: 检测到熬夜或长时间不学习
  - 庆祝进步: 检测到成绩提升时
  - 日常互动: 空闲时随机触发
- **情感识别**: 基于关键词和上下文判断用户情绪
- **反应系统**: 开心/担心/兴奋/安慰/鼓励 五种情绪响应

#### 3.1.3 任务系统
- **每日任务**:
  - 完成数学练习 (奖励理性能量)
  - 完成语文写作 (奖励诗意能量)
  - 完成科学实验 (奖励探索能量)
  - 完成英语对话 (奖励社交能量)
- **成就系统**: 连续学习天数、首次进化、属性满值等成就

#### 3.1.4 视觉反馈系统
- **粒子效果**: 学习时产生星星粒子
- **进化特效**: 能量满时触发进化动画
- **状态动画**: 宠物根据状态切换动作

### 3.2 数据流

```
用户行为 → 行为捕获 → 能量计算 → 状态更新 → UI渲染
                ↓
           AI对话触发 → LLM响应 → 对话显示
```

### 3.3 边界情况
- 能量上限: 单属性最大值100
- 进化条件: 达到60能量 + 满足等级要求
- 离线处理: 本地存储用户状态，下次上线同步

## 5. 技术栈

- **框架**: React 18 + TypeScript
- **样式**: CSS Modules + CSS Variables
- **动画**: Framer Motion
- **状态管理**: React Context + useReducer
- **AI对话**: Web Speech API (语音) + 模拟AI响应
- **构建工具**: Vite
